Digit Classification

Handwritten digit recognition is a common task in teh field of computer vision and pattern recognition, often used as an introductory project to explore various machine learning algorithms.This project built a classifier that can accurately recognize digits from pixel data using the k-NN algorithm.

Features

  • Data Preprocessing - Normalized and combined with corresponding labels for both training and testing datasets
  • Model Tuning - Use cross-validation to determine the optimal number of neighbors for the KNN classifier
  • Model Evaluation - Evaluated using accuracy metrics and a confusion matrix
  • Visualization - Providing insights into how well the model is recognizing patterns in the data

    • Libraries

  • Data manipulation and visualization: tidyverse
  • Model building, preprocessing, and evluation: tidymodels
  • k-NN algorithm: kknn
  • Plot: ggplot2
  • Controlling output: repr
