Campus Bullying Detection System

This project aims to assist Shandong Kaer Electronic Co., Ltd to develop an automated system capable of detecting bullying in campus conversations. By leveraging machine learning techniques, the system analyzes student dialogues and identifies potential instances of bullying. This enables schools to intervene promptly and effectively, ensuring a safer environment for students.

Key Features

  • Real-time bullying detection using audio surveillance
  • Natural Language Processing (NLP) for identifying harmful language in conversations
  • Integration with school monitoring systems for immediate alerts
  • Data analytics and reporting to track incidents

    • Technologies Used

  • Excel for model pre-processing and analysis
  • Chat GPT-4 and ChatGLM for data clustering
  • Real-time voice-to-text conversion
  • Machine Learning models for text analysis
  • Python, TensorFlow
