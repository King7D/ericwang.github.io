Image Similarity Finder and Downloader

This project aims to help machine learning developers quickly find their pre-processed image and accelerate the training process of checkpoints and models. The script downloads image, resizes them, and uses a pre-trained ResNet50 model to find and rename the cloest matching images in a directory. The script evaluates the similarities of imgaes, copies the matching images to a specified folder, and allows the model to train itself with these additional samples, thereby improving accuracy for future searches.

Features

  • Downloading images from the web
  • Resizing images for furhter processing
  • Using deep learning-based feature extraction and Nearest Neighbors to find and move the closest matching images
